Using jmxcmd with S1AS8PE

 

Author: lloyd.chambers@sun.com
Last updated: 03 June 2004

1. Introduction

Please familiarize yourself with the jmxcmd overview.

Here are step-by-step instructions for using jmxcmd with S1AS8 PE:

1. Download the following files into the same directory:

jmxcmd.jar
add-optional.jmxcmd

Then download the system-appropriate script:

Solaris: jmxcmd
Windows: jmxcmd.bat

Via jmxmp connector

The jmxmp connector has been removed.

Via Generic RMI connector (Appserver 8.1, June 3 build or later):

The RMI connector is standard; you can connect this way:

connect --host host --port 8686 --protocol rmi --user admin --password-file filename --jndi-name /management/rmi-jmx-connector name

Via HTTP/RMI connector

Connecting this way requires loading the JMXConnectorProvider class for Appserver via the add-provider command.

bash-2.05$ jmxcmd
# this only needs to be done once
jmxcmd > add-provider com.sun.enterprise.jmx.providers.SunJESAppserverProvider

jmxcmd > connect --host host --port 1234 --protocol sun-as-http --user admin --password-file filename name

Alternately, you can connect using RMI (note that the port differs from the HTTP port):

jmxcmd > connect --host host --port 8686 --protocol sun-as-rmi --user admin --password-file filename name
