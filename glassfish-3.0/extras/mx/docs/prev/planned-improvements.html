Idea Discussion
BSF MBean support Extend BSF support to include MBeans. This requires proxy generation and registration of the proxy within the BSFManager
Password encryption Encrypt the passwords within a password file (--password-file option for the connect command). They are currently cleartext.
RMI over TLS Support secure RMI via TLS, both in the built-in connector, and for the built-in Provider
Proxying results into local mbean server Creating local proxies for the results of various commands so that they can be used as MBeans locally.
I18N Cleanup of code to be I18N compliant.
