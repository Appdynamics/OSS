jcmd/jmxadmin build-info

Author: lloyd.chambers@sun.com
Last updated: 15 March 2004

1. Introduction

This document defines the built targets and their dependencies for jcmd and jmxcmd.

The source tree is rooted at com.sun.cli, which is the namespace granted and approved within sun for Java CLIs. The subtrees of com.sun.cli.jcmd and com.sun.cli.jmxcmd are the only subpackages the author is aware of at this time. There are three source directories: "src", "tests", and "optional".

2. Build targets

2.1 jcmd

Build jar: jcmd.jar

Source files:

src/com/sun/cli/jcmd/*

Dependencies:

JDK 1.4.x, jsse.jar

 

2.2 jmxcmd

Build jar: jmxcmd.jar

Source files:

src/com/sun/cli/jmxcmd/*

Dependencies:

JDK 1.4.x,
jsse.jar,
jcmd.jar*,
jmxri.jar*+,
jmxremote.jar*+,
jmxremote_optional.jar*+,
javax77.jar*+,
sasl.jar*+,
sunsasl.jar*+

* = incorporate the contents of this jar into the resulting built jar.
+ = these jars should be taken from JDMK 5.1

2.2.1 jmxcmd Java Enterprise System Application Server 8 support

Build jar: jmxcmd-jesas8.jar

Source files:

optional/cmd/com/sun/enterprise/s1as8/*
optional/providers/com/sun/enterprise/jmx/providers/SunOneHttpProvider.java

Dependencies:

jcmd.jar,
jmxcmd.jar,
jmxri.jar,
jmxremote.jar,
s1as-jsr160-client.jar

 

2.2.2 JUnit Test support

Build jar: jmxcmd-tests.jar

Source files:

tests/com/sun/cli/*

Dependencies:

jcmd.jar
jmxcmd.jar
jmxri.jar
jmxremote.jar
junit.jar*

* = incorporate the contents of this jar into the resulting built jar.

 

2.2.3 Optional MBeans

Build jar: jmxcmd-mbeans.jar

Source files:

optional/mbeans/*

Dependencies:

jcmd.jar
jmxri.jar
