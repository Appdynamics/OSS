Introduction

JCmd is an extensible command line framework and command-line shell, written in 100% Java.

JmxCmd builds upon the JCmd framework, retaining all its features, while adding a substantial set of new commands for managing, monitoring and viewing MBeans in JMX MBeanServers.

If you want to write your own custom command-line-interface (CLI), choose JmxCmd if you need JMX support; otherwise choose JCmd.

Please note that you may mix and match in any desired combination, including retaining none, some or all of the commands in either JCmd or JmxCmd (with the possible exception of the 'help' command, which is required).
