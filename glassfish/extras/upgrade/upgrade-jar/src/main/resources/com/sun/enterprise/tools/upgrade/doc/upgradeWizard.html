Upgrading Using the Upgrade Wizard

Use the Upgrade Wizard screen to specify the source and target installation directories and the master password of the target server if needed.

  1. In the Source Domain Directory field, enter the location of the existing installation from which to import the configuration.  You need to enter the domain directory. For example, <install-root>/domains/domain1.

  2. In the Target Domains Root Directory field, enter the location of the Application Server installation to which to transfer the configuration. You need to provide the domains root as the input to this field.
  3. Provide the master password of the source application server.
  4. Click Next button when done.
  5. The Upgrade Results panel is displayed showing the status of the upgrade operation.
  6. Click the Finish button to close the Upgrade Tool when the upgrade process is complete.
